Overview

"보기 좋은 숙소 예약 사이트는 없을까?"

긴 스크롤, 편의 기능 부재로 생긴 불편함을 개선했습니다.
새롭게 리뉴얼된 '여기어때'를 공개합니다.

여기어때 리뉴얼 프로젝트

Project Brief

  • 단순함
  • 신뢰할 수 있는
  • 안정적인

Problem

  • 최근 본 이력을 확인하기 어려움

  • 지역별 카테고리가 최하단에 위치

  • 호텔 객실 간의 비교가 불가능

  • 이벤트 탭 중복

  • 국내 숙소 검색 창 인기 검색 순위가 불필요

  • 인기 여행지 탭의 공간 차지

Approach

  • 최근 본 이력을 사이드바에서 확인 가능하게 기능 추가

  • 지역별 카테고리를 상단의 메뉴에서 확인할 수있도록 변경

  • 즐겨찾기 기능 추가

  • 상단의 메뉴에서 이벤트 메뉴 삭제 / 이벤트 배너 크기 조정

  • 국내 숙소 검색 창의 인기 검색 순위 삭제

  • 국내 인기 여행지와 해외 인기 여행지 탭을 하나로 합친 후, 버튼을 이용하여 전환 가능하게 변경

Main Target

취미가 여행인 사람들
타지역에 자주 머무르는 사람들

Sub Target

여행 주기가 1년 이상인 사람들

Persona

서진석

35세 | 남성

BIO

서울에 사는 영업사원 2년 차 진석 씨는 일주일에 한 번씩은 타지역으로 영업을 위해 출장을 갑니다. 적당한 가격에 깨끗한 숙소를 찾기 위해 여러 숙소를 확인해 봤는데 기록이 남지 않아 마음에 들었던 숙소 정보를 다시 찾는 것에 어려움을 느낍니다. 즐겨찾기 기능과 최근 기록 정보 기능의부재에 아쉬움을 가지고 있습니다.

Grievance

최근 기록을 확인할 수 없음, 자주 찾는 숙소의 확인이 불가능

Needs

즐겨찾기 기능

김한나

28세 | 여성

BIO

김한나 씨는 여행 경험을 글로 써 책을 출판하는 여행작가입니다. 직업 특성상 여행지와 가까우면서 글을 쓸 수 있는 환경이 마련되어 있는 숙소를 쉽게 예약하고 싶은데, 숙소들을 하나하나 비교하고 있으니 영 불편하기만 합니다. '여기어때'가 깔끔하고 보기 편안한 사이트인 것은 알고 있지만, 여러 편의 기능이 부족한 것도 사실입니다. 한나 씨는 해당 부분이 개선되었으면 하는 바람이 있습니다.

Grievance

숙소의 비교가 어려움

Needs

숙소나 방 등을 하나 고정하고 다른 숙소를 비교할 수 있는 기능

Project Benchmarking

  • trivago

    메뉴에 ‘최근에 본 상품’ 탭이 존재

    숙소 예약 사이트를 모아둔 곳임을 고려해도 숙소에 대한 정보가 부족

  • agoda

    찜, 장바구니 등 편의 기능 다수 존재

    화려한 색으로 시선 분산, 할인 표시가 과하게 많아 집중이 어려움

  • yanolja

    지역끼리 잘 정렬되어 있어 원하는 지역 선택이 가능

    정렬 기능이 너무 많은 비율을 차지하고 있음

01한눈에 볼 수 있는
검색 페이지

무작정 길기만 한 스크롤 페이지, 한번 보고 나면 잊어버리는 이력들. 최근 본 이력 탭으로 이전에 확인한 숙소들을 빠르게 찾아갈 수 있습니다. 개선된 필터 설정 탭으로 인해 보다 쉽게 검색 필터를 적용할 수 있습니다.

02찜목록으로
편리하게

같은 지역을 여러 번 방문할 때 유용한 찜 목록 기능으로 좋았던 숙소들을 두고두고 저장해둘 수 있습니다.

03한눈에
특가 확인하기

나열되어 있기만 했던 특가 목록을 보다 빠르게 확인할 수 있습니다. 원하는 키워드만을 확인하는 것도 가능합니다.

  • 여기어때 잘난체

    여기어때가 시각적 커뮤니케이션을 강화하기 위해 브랜드 아이덴티티를 담아 개발한 서체입니다. 라운드와 직선을 모두 담은 모음과 곡선으로 떨어지는 자음이 특징인 타이틀 전용 서체로 디자인 되었습니다.

  • 여기어때 잘난체 고딕

    여기어때 잘난체의 모듈과 자소를 유지하며 가독성을 높여 솔직하고 경쾌한 성격을 표현했습니다.

  • Pretnedard

    적합하지 않은 글꼴로부터 생기는 추가적인 소요를 줄이자는 데에서 시작하여, 깔끔함과 익숙함, 구태여 손을 보지 않고도 표준을 구성하는 것에 초점이 맞춰진 서체입니다.

Style Guide [ Logo ]

‘여기어때’라는 브랜드명을 명확하고 간결하게 표현한 로고입니다. 자체적으로 개발한 여기어때 잘난체를 변형한 형태로, 글자를 가로로 나열하여 모던함을 강조시켰습니다.

본 프로젝트의 목적은 브랜드의 사용자를 늘리기 위함이기 때문에 사용자들에게 비교적 익숙한 기존의 로고를 변형하지 않고 그대로 사용하였습니다.
