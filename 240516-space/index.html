About the project

'우주'라는 존재는 결코 모든 것을 알 수 없다는 특징 때문에 상당히 매력적인 소재입니다. 때문에 천문과 관련된 기사를 종종 찾아볼 정도로 우주를 좋아했습니다. 광활한 우주, 무한함, 끝이 어딘지 모름에도 차근차근 나아가는 인류 등을 저의 가능성이나 꾸준함 같은 요소와 연관을 지어 포트폴리오 페이지 컨셉으로 선택했습니다.

Project Goal

'우주'라는 컨셉에 걸맞는 역동적인 요소를 사용하면서도 전체 페이지의 흐름이 자연스럽도록 하였습니다. 자칫 과하게 보일 수 있는 컨셉을 모던한 레이아웃에 녹이며 보는 이에게 본인의 장점과 페이지의 컨셉이 잘 전달될 수 있도록 디자인하였습니다.

Approach

  • 섹션 간의 특징이 뚜렷하도록 디자인

  • ‘우주’에 걸맞은 역동적인 요소 추가

  • 모던한 레이아웃, 필요 요소를 제외하고 과감하게 제거

  • 우주
  • 꾸준함
  • 한계가 없는
  • 가능성

Point color

광활한 우주를 현대적으로 표현하기 위하여 흑백 위주의 색을 사용하였습니다. 때문에 포인트 컬러로는 무채색이 주는 답답함을 해소하기 위하여 청량감을 줄 수 있는 시원한 푸른색 계열을 선택했습니다.

Font

KBO 다이아고딕체

포인트를 줄 수 있는 요소가 포함되어 있으면서도 깔끔한 서체가 필요했습니다. 가독성 또한 중요한 요소였기 때문에 과하지 않은 디자인을 가지고 있으며, 다소 차가운 무드를 줄 수 있는 KBO 다이아고딕체 를 사용하였습니다.

INTRO

'우주'라는 컨셉이 가장 잘 드러나도록 표현하였습니다. 아이텐티티 슬로건과 자기소개 등을 심플하게 담았습니다.

PROJECT

개인 프로젝트와 팀 프로젝트를 한눈에 보기 쉽도록 탭 전환 요소를 사용하였습니다. 불필요한 요소들을 최대한 지우고 프로젝트에 집중할 수 있도록 하였습니다.

WORK

작업 목록을 동적으로 배치하였습니다. 상하, 또는 좌우로 움직이는 목록에 마우스 오버할 경우 상세한 설명이 나타나도록 하였습니다.

GRAPHIC

'우주'라는 키워드가 두드러지도록 그래픽 작업물들을 동적으로 배치하였습니다. 그래픽에 마우스 오버할 경우 작업물이 강조되면서 해당 작업물의 이름이 등장합니다.

ABOUT ME

강조를 위하여 기존의 무채색 배경 대신 포인트 컬러를 배경색으로 사용하였습니다. 때문에 자연스럽게 해당 섹션이 이어지도록 추가로 요소를 배치하였습니다. 내용은 행성이 공전하듯이 기준을 잡고 360도, 가독성을 위하여 버튼으로 회전할 수 있도록 하였습니다.
